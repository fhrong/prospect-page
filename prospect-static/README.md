# Prospect — Landing Page Estática (Centris Automation)

Versão 100% estática (HTML + CSS + JS puro). **Não precisa instalar nada.**

## Como usar

- **Abrir localmente:** dê duplo clique em `index.html`.
- **Publicar:** envie a pasta inteira para qualquer hospedagem estática
  (Netlify, Vercel, GitHub Pages, Hostinger, S3, ou um servidor Apache/Nginx).

## Estrutura

- `index.html` — página completa (HTML + CSS + JS inline)
- `assets/centris-logo.png` — logo da Centris
- `assets/hero-bg.png` — imagem de fundo do hero
